The situation

.......... 
  You have a file like this: a,b,c,d,e,f and you would like to turn it into a 
3-column table like this:
			a,b,c,
			d,e,f,
  The tool below makes it possible. As an example, copy and paste the above
text (that is, a,b,c,d,e,f) into the text box below.
  Then enter the delimiter, that is the character which seperates each column:
in our case it's the comma (,). However it can be any symbol or even a sequence
of characters. For example if our file was "a sep b sep c sep d sep e sep f sep"
the delimiter would be "sep" (without quotes).
  After, enter the number of fields per line: in other words, how many columns
each line should have, in our case 3. The tool allows up to 99 fields. If the
fields cannot match a value, they will return the default value "undefined".
  Finally press the "Go!" button. A new web page will open that will present the
previously unformated sequential file, into a table form with it's columns
separated by the delimiter (to return in this screen press "F5" in your keyboard).
Copy the content into a simple word processor (i.e. notepad or equivalent) save
it as a text file and import it into your spreadsheet. You 're done!
